## Project Description

An object-oriented game where the player faces three different beasts, each with unique elemental powers (Earth, Water, Fire). The player is provided with three types of potions and must use the hints strategically to determine which potion to use to defeat each enemy. The game demonstrates interactions between players and monsters, including bosses with higher difficulty.

## System Features

- Player encounters three different elemental beasts.
- Each beast has unique strengths and weaknesses.
- Player can choose from three different potions.
- Hints are provided to help choose the correct potion.
- Boss enemies with higher difficulty increase challenge.
- Game demonstrates object-oriented interactions between classes.

## OOP Concepts Used

- **Encapsulation**: Attributes are private/protected, accessed via properties or methods.  
- **Abstraction**: Classes expose only necessary details for interaction.  
- **Inheritance**: Enemy and Boss classes inherit from a common Monster class.  
- **Polymorphism**: Methods are overridden for different monster behaviors.
